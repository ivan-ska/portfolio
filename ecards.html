eCards Premium Landing page

eCards is a fintech platform providing high-performance virtual cards for media buying and affiliate marketing. To support the launch of their Premium tier I designed and developed a dedicated landing page that highlights the service’s priority onboarding and exclusive banking features for professional teams.

As the sole designer and developer on the project I handled the entire process—from information architecture and visual identity to custom-coded implementation. I built a polished site with smart responsiveness using a vibecoding approach.

At the project’s start, I worked with the client to define the core attributes and visual direction of the Premium tier. We established a set of brand descriptors and personality sliders to align our vision and create a clear "north star" for the landing page.

This framework acted as a constant reference point throughout the process. It helped us stay on track with visual style, information delivery, and text vibe.

Instead of relying on stock libraries, I developed a custom visual language for the page. Every icon and graphic was created from scratch to match the specific context of the copy. This ensures that the visuals act as a functional part of the narrative, helping the user digest complex fintech features rather than just filling space.

On the implementation side, I focused on system-wide consistency. Even within a fast-paced "vibecoding" workflow, I maintained a strict logic for corner radii, spacing, and hover states. The code is built on a clean, reusable structure rather than a collection of random styles.

The final result is a clean, production-ready landing page. It doesn't just describe a premium service but communicates that value through its execution. I made sure the site adapts to any screen size so it feels just as solid on mobile as it does on a desktop. I handled the entire project on my own, taking it from the initial text edits and structure discussions all the way to the final public launch.
